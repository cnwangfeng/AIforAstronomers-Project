# Data 目录说明

本目录保存教材配套数据、数据说明和下载脚本。数据目录必须服务于教材和 notebooks 的可复现性。

建议目录结构：

```text
data/
  small/      # 可直接随仓库发布的小型教学数据
  external/   # 外部下载数据的本地缓存位置，默认不提交大文件
  scripts/    # 数据下载、校验和预处理脚本
```

数据使用原则：

- 小数据可以提交到仓库，但应尽量控制体积。
- 大数据不直接提交，只提供下载链接、脚本、校验方式和预处理说明。
- 每个数据集必须说明来源、许可证、下载日期、字段含义、单位、样本量和适用章节。
- notebook 中不得依赖未公开、未说明或只存在于个人电脑上的私有数据。

数据集说明模板：

```text
名称：
来源：
许可证：
下载日期：
文件：
字段与单位：
预处理步骤：
适用章节/notebook：
注意事项：
```
